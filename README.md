## Allure Tanquintic - Blog App

### What I was able to accomplish:

- User model
- CRUD functionalities
- JWT authentication

### How to run

## Install packages

```shell
npm install
```

## Seed

```shell
node bin/seed.js
```

## Run app

```shell
npm run dev
```

Credentials:

```shell
username: testuser
```
