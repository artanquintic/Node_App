## Allure Tanquintic - Blog App

### What I was able to accomplish:

- User model
- CRUD functionalities
- JWT authentication
- Third-party API integration (NewsAPI)

### Things that I would like to add if given more time:

- Tests
- Fix error messages

## How to run

### Install packages

```shell
npm install
```

### Seed

```shell
node bin/seed.js
```

### Run app

```shell
npm run dev
```
